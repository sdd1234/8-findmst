기본적으로 100번의 탐색을하는데 
100번을 하면서의 순차탐색의 평균적인 시간복잡도는 3/4xn번이다
100으로 계산한다면 75번의 비교를한다
하지만 이진탐색에서는 log n의 비교를 하는데 이는 퀵소트로 정렬을 한 상태이기 때문이다

확실히 비교를 후자가 많이 하기는 한다
